# Security Policy

This repository is for authorized security testing only.

Do not scan systems unless you have written or clearly delegated permission for the exact target and scan type. Active scans can send attack-like payloads and may affect fragile applications.

Generated reports are ignored by Git because they may contain sensitive details. Review them before sharing, and redact secrets, cookies, private URLs, or personal data.

If you find a vulnerability in this toolkit, open a private disclosure channel with the repository owner rather than publishing exploit details first.
